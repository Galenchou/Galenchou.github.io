发表于 2017-11-28

服务器重启后的问题

由于百度云上的服务器今天到期，所有公司的几个网站都访问不了。因此在续费成功以后，出现了服务器上面的nginx和tomcat服务器都停止运行了，然后在重启后的nginx的时候出现了一个问题

然后再网上查看了此问题，大概就是window服务器再重启的时候，会默认打开自己的iis服务器，这个服务器会占用80端口，所以会导致ngixn启动不了，随机再服务器里面设置关闭iis服务器就ok.
